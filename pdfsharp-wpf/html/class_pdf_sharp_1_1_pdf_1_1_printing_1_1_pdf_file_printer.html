Detailed Description

A wrapper around Adobe Reader or Adobe Acrobat that helps to print PDF files. The property AdobeReaderPath must be set before the class can be used for printing. The class was tested with Adobe Reader 7.0.7. If this stuff does not work, please don't write me mails! If you enhance this class, please let me know.
